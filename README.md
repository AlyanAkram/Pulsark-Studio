# Pulsark Studio

Pulsark Studio is a lightweight, fast, and extensible code editor built in **Rust** as part of the Pulsark ecosystem.

It is designed to evolve into a full-featured development environment focused on performance, simplicity, and extensibility.

---

## 🚀 Current Status

Pulsark Studio is currently in **active development (early prototype stage)**.

Some core features are already functional, while others are still in progress.

### ✅ Working Features
- File explorer sidebar (basic implementation)
- File opening and navigation
- Integration with external system/terminal output (temporary)

### 🚧 In Progress
- Scroll support for file explorer (needed for large projects)
- Improved UI consistency and layout handling
- In-app integrated terminal (currently placeholder / coming soon)
- Performance and stability improvements

---

## 🧠 Vision

Pulsark Studio aims to become a powerful yet lightweight IDE that balances:
- Speed of native applications (Rust-powered)
- Clean and modern UI/UX
- Extensibility for plugins and tools
- Developer-focused workflow optimization

---

## 🛠 Tech Stack

- **Language:** Rust
- **UI Framework:** (custom / in-house UI layer)
- **Architecture:** Modular, performance-first design
